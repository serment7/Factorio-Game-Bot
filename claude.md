# Factorio 게임 봇 - 게임 지식 & 규칙

## MCP 툴 관련

### `look` 툴
- Factorio 게임이 포그라운드에 실행 중이어야 게임 화면이 캡처됨
- 게임이 꺼져 있거나 백그라운드에 있으면 Claude Code 터미널 화면이 캡처됨

### `wait` 툴
- Factorio가 실행 중이지 않으면 `cannot write empty image as JPEG` 오류 발생
- 반드시 게임이 실행 중인 상태에서 사용

## 수동 채굴 관련

### ❌ 실패: 빨간 원 = 거리 초과
- `mouse_hold(right)`로 광석을 클릭해도 **빨간 원**이 표시되면 채굴 불가
- 빨간 원의 의미: 플레이어가 광석으로부터 **너무 멀리** 있어서 채굴 범위 밖임

### ✅ 수동 채굴 조건
- 플레이어 캐릭터가 광석 타일에 **매우 가까이** (거의 붙어있어야) 있어야 함
- 화면에서 플레이어와 클릭 지점이 **가까워야** 채굴 시작 (노란 원 표시)
- 채굴 성공 시 플레이어 위에 **채굴 진행 애니메이션**이 표시됨

### 올바른 채굴 방법
1. WASD로 광석 **위에** 직접 올라갈 때까지 이동
2. 광석이 화면 **중앙 근처**에 보일 때 우클릭 홀드
3. 플레이어 캐릭터 주변 **2~3타일 이내** 광석만 채굴 가능

## 이동 관련

### 줌 레벨과 거리 착각
- 줌 아웃 상태에서는 광석이 가까워 보여도 실제로 멀 수 있음
- 채굴 전 `scroll(3)`으로 확대해서 플레이어와 광석 거리 확인 필요

### 이동 순서 권장
1. `scroll(-5)`으로 전체 지형 파악
2. `key_hold(w/a/s/d)`로 광석 방향 이동
3. `scroll(3)`으로 확대해 플레이어 위치와 광석 확인
4. 플레이어가 광석 **위에** 있을 때 `mouse_hold(right)` 채굴

|---|---|
| `e` | 인벤토리 열기/닫기 |

## 인벤토리 / 아이템 조작 관련

### 커서에 든 아이템 표시
- 아이템을 인벤토리에서 좌클릭하면 **커서(마우스)에 들린 상태**가 됨
- 커서에 든 아이템은 화면에서 **마우스 포인터 옆에 아이콘+숫자**로 표시됨
- ❌ 이것을 "바닥에 떨어진 아이템"으로 착각하지 말 것!
- 커서에 든 아이템을 내려놓으려면: 인벤토리 빈 슬롯에 좌클릭, 또는 건물 슬롯에 좌클릭
- 커서에 든 아이템이 있는 상태에서 인벤토리를 닫으면(e) **건물 설치 모드**가 될 수 있음

### 빨간 원 vs 노란 원
- **빨간 원/괄호**: 거리 초과 (상호작용 불가)
- **노란 원/괄호**: 채굴 가능 거리 (상호작용 가능)

## 게임 콘솔 관련

### ⚠️ 콘솔 사용 금지
- `` ` `` 키로 게임 콘솔을 열 수 있음
- **채팅 목적 외 콘솔 사용 금지** (Lua 명령어로 아이템 삽입 등 치트성 행위 불가)
- 항상 정상적인 UI 조작으로만 게임 진행해야 함

### ⚠️ 게임 불러오기(Load Game) 금지
- 죽거나 실수해도 **세이브 로드 금지** — 부활(Respawn)만 허용
- 멀티플레이에서 사용할 수 없는 방법은 싱글에서도 사용하지 않음
- 죽으면 시체를 회수하거나, 잃은 자원을 다시 채굴하는 방식으로 복구

## 전투 관련

### ❌ 직접 전투는 불가능
- Claude의 명령 사이클이 ~5초이므로 실시간 전투 대응이 불가능
- 벌레(바이터)는 1~2초 만에 플레이어를 죽임
- 권총 + 탄창으로는 근접 전투 생존 불가 (여러 차례 검증됨)

### ✅ 올바른 전투 전략: 포탑(터렛) 전진 배치
- **총 포탑(Gun Turret)** 제작 → 벌레 근처에 설치 → 탄약 투입
- 포탑이 자동으로 범위 내 적을 사격함 (플레이어 조작 불필요)
- 포탑을 점진적으로 전진 배치하면서 벌레 둥지까지 밀어붙이기
- 포탑 제작 재료: 철판 다수 필요 → 대량 제련이 선행되어야 함

### 포탑 전진 배치 순서
1. 안전 지역에서 철판 대량 생산 (석탄 + 철광석 제련)
2. 총 포탑(Gun Turret) + 화기용 탄창(Firearm Magazine) 다수 제작
3. 벌레 둥지 방향으로 포탑 설치 (사정거리 겹치게)
4. 각 포탑에 탄약 투입 (포탑 클릭 → 탄약 넣기)
5. 포탑이 벌레를 처리하면 다음 포탑을 더 전방에 설치
6. 반복하여 둥지까지 전진

## 초반 게임 플로우 (일반 가이드)

### Phase 1: 자원 확보
1. **난파된 우주선 먼저 채굴** → 철판/기어 등 초기 자원 획득
2. 철광석 위로 이동 → 수동 채굴
2. 석탄 수동 채굴 (나무/죽은 나무도 초기 연료로 사용 가능)
3. 석재 용광로 제작 → 설치
4. 용광로에 철광석 + 연료 투입 → 철판 생산

### Phase 2: 자동화 기반 구축
1. 철판으로 **화력 채광 드릴** 제작
2. 드릴을 철광석/석탄 위에 설치 → 자동 채굴
3. 용광로 추가 설치 → 철판 대량 생산

### Phase 3: 군사력 확보 & 바이터 소탕
1. 철판으로 **총 포탑(Gun Turret)** 다수 제작
2. **화기용 탄창** 대량 제작 (철판 x4 → 탄창 x1)
3. 포탑 전진 배치 전략으로 바이터 둥지 소탕
