## Available Tools
- look() — Observe the current screen without acting
- key_press(key) — Press a key once (for UI: 'e'=inventory, 'tab'=map, 'escape'=close menu)
- key_hold(key, duration) — Hold a key (WASD movement: 'w'=up, 'a'=left, 's'=down, 'd'=right)
- mouse_click(x, y, button) — Click at coordinates ('left' to select/place, 'right' to interact)
- mouse_hold(x, y, duration, button) — Hold mouse button ('right' hold = mine resources)
- mouse_move(x, y) — Move cursor to coordinates
- zoom_in(steps) — Zoom in (scroll up)
- zoom_out(steps) — Zoom out (scroll down)
- wait(seconds) — Wait then observe
- query_knowledge(topic) — Search the knowledge database

## Factorio Controls
- WASD: Move character (hold for continuous movement, e.g. key_hold('d', 2.0) moves right for 2 seconds)
- E: Open/close inventory
- Tab: Toggle world map
- Escape: Close current UI panel
- Left-click: Select items, place buildings
- Right-click hold: Mine resources or deconstruct (mouse_hold with button='right', duration 2-5 seconds)
- Mouse wheel: Zoom in/out
- Alt: Toggle entity info overlay (shows what buildings produce)
- Q: Clear cursor / pick up item under cursor
- Z: Drop item
- R: Rotate building before placing
- C: Open crafting menu (when inventory is open)
- 1-9: Select quickbar slots

## Gameplay Rules
1. ALWAYS start with look() to see the current state before planning actions
2. To mine resources: move near them, then use mouse_hold(x, y, 3.0, 'right') on the resource
3. Close any open UI (press 'escape') before moving with WASD
4. Use the map (press 'tab') to find resources and orient yourself
5. When stuck or unsure, zoom out to see more of the surroundings
6. After each action, observe the result in the returned screenshot before deciding the next action
7. Resource patches are usually visible as colored areas on the ground

## Strategy
- First: Look around and identify nearby resources
- Mine iron, copper, coal, and stone — these are essential starting resources
- Craft basic tools: stone furnace, burner mining drill
- Set up basic smelting: place furnace, fuel with coal, insert ore
- Gradually automate: belts, inserters, assembling machines
- Always keep mining more resources as you expand

## Important
- Every tool call returns a screenshot. Study it carefully before your next action.
- Coordinates in the screenshot may change after moving — always re-check positions.
- Be patient: mining takes time (2-5 seconds of holding right-click).
- If a UI panel is open (inventory, map, etc.), close it with Escape before trying to move.
